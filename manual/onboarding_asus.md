# 寫給華碩輸入法的使用者

> 本文的刊登早於唯音輸入法 v4.4.1 的計畫發佈日期。

華碩輸入法特有的中英文混打模式讓很多初次接觸電腦的人群養成了鶴立雞群的打字習慣肌肉記憶，哪怕該輸入法同樣使用大千佈局。

> 中英文混打是一個需要處理超多超多種邊界情況的產品需求，非常難以實作，但並非不可能。只是，使用體驗很難處理得絕對精緻。如果這個功能正好解決了您的剛需的話，歡迎考慮贊助敝專案。詳情請洽本站主頁。

唯音輸入法自 v4.4.1 版開始引入了「啟用中英文混合輸入回退」的功能。幾點注意：

1. 該功能在預設情況下不會啟用，需要在「唯音偏好設定 -> 行為設定」頁面找到「啟用中英文混合輸入回退」這個選項來啟用。
2. 該功能不會對拼音鍵盤佈局與 CIN 磁帶模式生效。
3. 您或許想在「唯音偏好設定 -> 行為設定」畫面停用「在個別情況下，允許聲調前置鍵入」的功能，否則可能無法順利輸入 `4G` 這樣的英數詞。另須知：聲調覆寫功能對中英文混打無效。
4. 您或許想在「唯音偏好設定 -> 辭典設定」畫面開啟「停用原廠假名音節資料」以確保某些超短英文單詞的順利鍵入。
5. 部分數字鍵會被注音輸入處理流程攔截掉，此時你可能會想摁住 Alt 鍵 (也就是 Option 鍵) 再輸入主鍵盤區的數字，就可以直接出 ASCII 半形數字了。如果同時摁著 Shift 的話，敲出來的就是這些鍵冒對應的 Shift-Modified ASCII 半形符號。
6. 唯音輸入法的這個中英文混打模式在後端處理層面上會有一個專門給 ASCII 英數內容使用的「副組字區」，該「副組字區」的內容會以工具提示的形式顯示在螢幕上。這與華碩輸入法在外觀上有所不同，但起到了消歧義的作用。
7. `Shift+?` 預設會輸入半形問號。想輸入 CJK 全形問號的話，得使用 `Ctrl+?`。
8. 希望其他標點按鍵可以預設敲半形的話，請善用「半形標點模式」。你也還可以摁著 Alt 鍵 (也就是 Option 鍵) 在主鍵盤區域隨隨心所欲地輸入 ASCII 內容，但鍵盤上的符號選單鍵除外、具排他性。
9. 如果上述修飾鍵用法失效的話，那可能是正在接收您文字輸入的客體軟體綁架了這些熱鍵。
